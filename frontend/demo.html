数据安全

安全第一，隐私至上

多层安全防护，全方位保护用户数据

🔐

JWT 身份认证

基于 JSON Web Token 的无状态认证，安全可靠

🛡️

XSS 防护

输入过滤中间件 + CSP 安全头，防止跨站脚本攻击

🔒

SQL 注入防护

SQLAlchemy ORM 参数化查询，从根本上杜绝 SQL 注入

🎫

CSRF 防护

CSRF Token 机制，防止跨站请求伪造攻击

⏱️

接口限流

登录接口 5 次/分钟，普通接口 60 次/分钟，防止暴力攻击

🔑

密码加密

bcrypt 加密存储，即使数据泄露也无法还原明文密码
